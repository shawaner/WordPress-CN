
# WordPress-CN

## 简介

这是一个用于辅助阅读学习 WordPress 源代码的项目。

该项目专注于 WordPress 源码分析，在源代码中直接加入作者对 WordPress 设计以及程序逻辑的理解说明（以中文注释的形式呈现），并保留原文注释和代码逻辑。

中文注释风格：

- PHP 采用 # 号开头的注释风格；
- HTML 内容仍使用 &lt;!-- comments --&gt; 形式书写。

该项目中的 WordPress 源代码基于 v4.9.6 版本，添加或修改的内容仅用于辅助阅读，不会修改代码逻辑。
